# Mess Management System

Mess Management System is a college project designed to manage hostel mess operations efficiently. It helps students and administrators handle meal tracking billing menu management and feedback digitally.

## Features
- Student registration and login
- Daily and weekly mess menu
- Meal attendance tracking
- Billing and payment management
- Feedback and complaint system
- Admin dashboard

## Technology Used
Frontend: HTML CSS JavaScript
Backend: Node.js / Python / PHP
Database: MySQL / MongoDB

## Purpose
The system reduces manual work improves transparency and minimizes food wastage in college mess operations.
